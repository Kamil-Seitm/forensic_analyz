# GUI_flesh_Fast_IR_analyz

Сценарий для флешки: запускаем FastIR, сразу анализируем вывод и оставляем отчёт на том же носителе.

## Что кладём на флешку
- `FastIR_x64.exe` (оригинальный сборщик артефактов).
- Скомпилированный GUI-анализатор (`fastir_gui.exe`).

## Сборка GUI в `.exe` (на машине разработчика под Windows)

```bash
python -m venv .venv
.venv\Scripts\activate
pip install .
pip install pyinstaller
pyinstaller fastir_gui.spec
```

После сборки готовый файл: `dist\fastir_gui.exe`.

## Как использовать на целевой машине

1. Копируем на флешку:
   - `FastIR_x64.exe`
   - `fastir_gui.exe`
2. Вставляем флешку в целевую машину и запускаем `fastir_gui.exe` от имени администратора.
3. В поле **FastIR_x64.exe** указываем путь на флешке (по умолчанию — рядом с GUI).
4. В поле **Корень флешки/папка вывода** указываем букву флешки (например, `E:\`).
5. Нажимаем **Старт**. Приложение:
   - запускает FastIR с параметром `-o <флешка>/fastir_<timestamp>/fastir_raw`;
   - прогоняет эвристики, сохраняет отчёт (`fastir_report.txt` или `.json`);
   - по желанию кладёт ZIP со всеми данными (`fastir_bundle.zip`).
6. После завершения пути до артефактов и отчёта отображаются в статусной строке и окне.

## Быстрый сценарий для “втыкаем и забираем”

- Делаем ярлык `fastir_gui.exe` в корне флешки.
- Запускаем `fastir_gui.exe` — через пару минут на флешке появляется папка
  `fastir_YYYYMMDD_HHMMSS` с сырыми CSV/JSON и отчётом.
